# AI Tools Hub - Vite + React + Tailwind Starter

Resumo rápido:
- Um site moderno que lista, compara e recomenda ferramentas de IA (texto, imagem, vídeo, voz).
- Monetização: anúncios (AdSense), links de afiliados, newsletter.
- Conteúdo em inglês para atrair tráfego dos EUA e ganhar em dólar.

O que tem aqui:
- Frontend em React (Vite) com TailwindCSS.
- Página principal com cards de ferramentas, busca, e páginas estáticas.
- Código pronto para abrir no VS Code e rodar localmente.

Como rodar (no seu computador):
1. Instale Node.js (versão 18+ recomendada).
2. No terminal, abra a pasta do projeto.
3. Rode:
   npm install
   npm run dev
4. Abra http://localhost:5173
